  * Proyecto desarrollado por Ángel Martínez Otero - 2023/2024.
  * Proyecto Baraja de Cartas | Desarrollo de Aplicaciones Web (1ºDAW).
  * Consiste en la implementación de una baraja de cartas española, permitiendo intercambiar cartas de manera interactiva.
  * El objetivo principal fue aplicar conceptos fundamentales de programación orientada a objetos para gestionar una baraja, realizar intercambios de cartas y analizar cuántas cartas cambiaron de posición tras barajar.
